TOL_SRPG

===

タクティクスオウガライク(TOL)なSRPGをC#で目指して制作した
プロジェクトデータと、関連ファイルの一部です(※ビルド、動作確認に十分なもの)

## 開発環境

Windows7 - VisualStudio2015 C#
Python 3.5.1 (バッチ処理用、ビルドには依存しない)

## 利用しているもの
+ DXライブラリ(C#用)
+ MigMixフォント

## 同封されないデータ

下記のデータは同梱していません
+ project/TOL_SRPG/bin/Debug/sound/

## ライセンスについて

### 当該

TOL_SRPGの著作権は、Ｍ−ｇｅｎが保持します。
著作権保持者は、利用についていかなる責任をおいません
著作者保持者は、修正の義務をおいません
再配布、改変とその配布を制限しません

当該のライセンスの表記は不要です

### 依存

依存するライセンス、著作権表記は
document/ReadMe.txt をご確認ください

MigMixフォントは、配置された場所に、
ライセンスが記載されたファイルもそのまま配置しています

DXライブラリに関わるもので、記載が必要なものが有ります
